1. Вычисления и контейнеризация:
EC2: виртуальные серверы для запуска Rust-приложений
ECS/EKS: оркестрация контейнеров для микросервисной архитектуры
Lambda: serverless функции для обработки транзакций и событий
Fargate: serverless контейнеры
ECR: registry для хранения Docker образов

2. Базы данных:
RDS: реляционные БД для финансовых данных и ACID транзакций
DynamoDB: NoSQL для высокочастотной торговли и real-time данных
DocumentDB: для хранения сложных финансовых документов и JSON структур
ElastiCache: кэширование для низкой латентности

3. Безопасность:
IAM: управление доступами
KMS: управление ключами шифрования
GuardDuty: обнаружение угроз
Secrets Manager: безопасное хранение API ключей и токенов
AWS WAF: защита веб-приложений от атак

4. Сеть и API:
API Gateway: управление API
CloudFront: CDN для глобального доступа
VPC: изолированные сетевые среды
Load Balancers: распределение нагрузки (ALB/NLB)
Auto Scaling: автоматическое масштабирование

5. Данные и аналитика:
Kinesis: потоковая обработка данных
Redshift: хранилище данных для аналитики
Glue: ETL для обработки больших объемов данных
QuickSight: бизнес-аналитика

6. Интеграции и сообщения:
EventBridge: событийно-ориентированная архитектура
SQS: очереди сообщений для надежной обработки
SNS: уведомления и pub/sub паттерн

7. Мониторинг и логирование:
CloudWatch: метрики и мониторинг
CloudTrail: аудит API вызовов
AWS Config: compliance и конфигурационный аудит
