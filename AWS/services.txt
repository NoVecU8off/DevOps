1. Вычисления и контейнеризация
EC2: виртуальные серверы с dedicated hosts для regulatory compliance
ECS/EKS: оркестрация контейнеров с AWS Fargate для serverless containers
Lambda: serverless функции с VPC support для secure transaction processing
Batch: managed batch computing для end-of-day financial calculations
ECR: private registry с vulnerability scanning и image signing
App Runner: fully managed containerized applications с auto-scaling

2. Базы данных и хранение
RDS: managed PostgreSQL/MySQL с Multi-AZ deployment и read replicas
Aurora: cloud-native database с global database для cross-region replication
DynamoDB: NoSQL для high-frequency trading data с DAX for microsecond latency
DocumentDB: MongoDB-compatible для complex financial documents
ElastiCache: Redis/Memcached caching с cluster mode и encryption
QLDB: immutable ledger database для audit trails и transaction logs
Timestream: time-series database для market data и IoT financial sensors
